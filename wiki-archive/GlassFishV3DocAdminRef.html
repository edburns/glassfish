Audience

The Domain File Format Reference (formerly Administration Reference) is for only the most advanced, XML-savvy administrators.

Purpose and Description

The Domain File Format Reference describes the domain.xml file, which contains most of the Enterprise Server configuration.

Statement of Work

This section describes in detail for each milestone the work that is needed on the document, such as changes, addition of new material, relocation of existing material, and removal of existing material. Work might result from new features, changes to existing features, bug fixes, RFEs, and editorial enhancements.

FCS Statement of Work

For FCS, all of the GFv2 elements will be restored except those related to clusters.

Elements Documentation Impact Description of Change
applications, system-applications, and children MAJOR Entirely new element tree but with much functionality moved from GFv2
resources and children MAJOR Updates for jdbc-connection-pool and connector-connection-pool, new work-security-map
network-config, http-service, and children MAJOR Entirely new element tree but with much functionality moved from GFv2
All other elements Minor Mostly reinstatement of GFv2 elements

Overall Content Outline

Most of the content consists of the XML elements described in alphabetical order.
