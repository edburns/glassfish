Isn't there an easier way to install GlassFish? What about the IzPack installer?

This FAQ is based on this thread in
which the cumbersome GlassFish install process was discussed and suggestions were made for improvements in
GlassFish v3 Requirements. Issues captured on these v3 requirements pages will also be moved to the [Issue
Tracker|https://glassfish.java.net/servlets/ProjectIssues] and tracked there. Also see How do I file an RFE for GlassFish v3?.

It was noted in the course of the discussion that the IzPack installer
from Julien Ponge offers one-step installation and greatly simplifies the installation process. The IzPack installer
was originally announced in this thread and
this blog post. Solaris support
and other enhancements have since been added. Also see this IzPack showcase that features GlassFish v2.
