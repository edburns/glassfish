GlassFish v3 技术预览 2 (Technology Preview 2, TP2) 是什么？

GlassFish v3 技术预览 2 (Technology Preview 2, TP2) 是正在开发中的 GlassFish v3 Application Server 的里程碑发行版。

通过 TP2 发行版，您可以及早了解 GlassFish v3 及其模块化、轻量级、可扩展的体系结构。TP2 不是完整的功能完善的应用服务器，因此*不适用于生产部署*。它需要进一步的试验和探索，而这正是我们希望您做的。使用代码试一下，看看 GlassFish v3 中所采用的新方法，然后将您的想法[告诉我们]（甚至还可以亲身参与）。

有关此发行版所含内容的更多信息，请参见 GlassFish v3 TP2 中包含什么内容？。有关下载信息和其他有用的资源，请参见如何获得和开始使用 GlassFish v3 TP2？。
