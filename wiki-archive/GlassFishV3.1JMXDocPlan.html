Audience

JMX is for advanced administrators who want to use the API to monitor and manage a server configuration. AMX is a GlassFish-specific subset of the JMX API, which complies with standards such as JSR 77.

Summary of the Feature From a User's Perspective

What the user mainly needs to know is how to navigate the GlassFish Server configuration using a standard JMX client. The easiest way to accomplish this is to explain the mapping between the asadmin get/set/list dotted name hierarchy and the AMX object name hierarchy.

Availability

JMX/AMX is available in all GlassFish Server distributions.
