GlassFish 真的免费吗？

*是的！*GlassFish 应用服务器是开源的，完全免费使用。Sun GlassFish Enterprise Server（以前称为 Sun Java System Application Server）是 Sun 的商业版本的 GlassFish，它具有不同的安装程序，也完全免费使用。

如果 GlassFish 和 Sun 的商业版产品都是免费的，那么 Sun 怎么赚钱呢？一句话，靠服务赚钱。Sun 通过销售 Sun GlassFish Enterprise Server 品牌的服务（即支持、咨询和培训）来赚钱。

将开放源代码产品化。

如果您打算进行生产，并希望购买支持（或将来购买支持），Sun 建议您在生产系统上安装 Sun GlassFish Enterprise Server。Alexis 写了一篇极好的文章，介绍支持的益处。最重要的几点是，及时而准确地响应 SLA 问题，访问持续发展的分支以及保证金。保证金是支持合约（而不是 Sun GlassFish Enterprise Server 组件本身）所带来的好处。Arun 还在 3 Steps for Successful GlassFish Deployment（三步成功实现 GlassFish 部署）中说明了订阅的益处。
