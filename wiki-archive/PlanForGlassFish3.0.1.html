Release Scope / Goals

This is a follow-on minor update release for GlassFish Server Open Source Edition 3.0 that essentially contains changes related to first level of Oracle integration of acquired Sun products - also referred to as "100 day releases". These changes are being made to the open source project and will be consumed downstream by the Oracle branded commercial release of Oracle GlassFish Server. See GlassFish Server Open Source Edition Roadmap slides below. This release addresses the following:

  • Branding changes
  • Multi-Lingual releases
  • Critical Bug fixes
