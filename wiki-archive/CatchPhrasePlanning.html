GlassFish Catch Phrase Planning

Overview

We need to select at least one "catch phrase" for use on GlassFish buttons and other materials.

Points to consider:

  • MUST NOT imply that GlassFish is just an app server
  • MUST be short enough to fit into small web button images
  • SHOULD be catchy enough to make people want to learn more about GlassFish
  • SHOULD give people some notion of what GlassFish is

Examples from other projects:

  • Java: Run Desktop Apps Faster with Java
  • NetBeans: Simple, Intuitive IDE
  • Solaris: The Most Advanced OS on the Planet

Candidate Phrases:
