如何在我的 Web 应用程序中激活符号链接？

要在您的 Web 应用程序中激活符号链接，请在您的 Web 应用程序的 <code>sun-web.xml</code> 中指定此属性：

<sun-web-app>
    <property name="allowLinking" value="true"/>
  </sun-web-app>

此属性的默认值为 <code>false</code>。
或者，您也可以在 <code>domain.xml</code> 中的 <code><virtual-server></code> 级别指定此属性，此时它将应用于虚拟服务器上部署的所有 Web 应用程序，除非被某个 Web 应用程序的 <code>sun-web.xml</code> 覆盖。
