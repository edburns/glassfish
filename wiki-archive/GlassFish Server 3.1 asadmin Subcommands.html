This page lists all asadmin subcommands that are to be exposed to users in GlassFish Server 3.1. The package in which each subcommand is delivered is listed.

The Source column indicates one of the following sources:

  • 3.0.1. The subcommand was supplied in the 3.0.1 release.
  • v2. The subcommand was supplied in v2 but not in 3.0.1. The subcommand is to be forward-ported from v2 to 3.1.
  • 3.1. The subcommand is new in the 3.1 release.

The table summarizes the differences from 2.1 and 3.0. The following differences have been ignored in this summary:

  • Differences in how the target option is printed:
    • In 3.1, the target option is printed as --target <target(default:server)>.
    • In 3.0, it is just --target <target>.
  • The correction of the misspelt "defalt" in the redeploy usage message
  • The addition of the target option or operand to many subcommands in 3.1
  • Differences in subcommands from 2.1 that were already in 3.0- that is, only the 2.1 subcommands that have been brought forward to 3.1 are compared
  • Changes to the names of operands (not options)
