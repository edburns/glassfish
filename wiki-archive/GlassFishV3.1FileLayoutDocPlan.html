Audience

This feature is aimed at system administrators and application developers.

Summary of the Feature From a User's Perspective

This project defines product file layout used to deliver the content of GlassFish Server Open Source Edition 3.1 as a standalone product distribution in IPS format that can integrated in Update Center 2.x.

Availability

This feature is available in all GlassFish Server distributions.

Changes to Books

Multiple/All Books Changes

The default product installation parent directory is changing in GlassFish 3.1 from glassfishv3 to glassfish3, which requires minor updates to many documents in the GlassFish 3.1 documentation set. Specifically, all instances of glassfishv3 must be changed to glassfish3 in large number of book component and shared entity files. Moreover, for GlassFish installations on Microsoft Windows, the path names will have to be further modified, as follows:

  • system drive:\glassfish3 (Windows XP platform)
  • %UserProfile%\glassfish3 (Windows Vista and newer Windows platforms)

The following table lists all the book files and entity files in which these changes must be made.
