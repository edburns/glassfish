Audience

This feature is aimed at system administrators.

Summary of the Feature From a User's Perspective

The Clustering Infrastructure feature enables basic functionality for the creation and management of clusters of multiple GlassFish Server instances.

This feature also provides the clustering infrastructure that is required to support other features such as session replication, load balancing, and failover.

Availability

This feature is available in the Web Profile distribution and the Full Platform distribution.
