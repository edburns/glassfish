Audience

The Application Deployment Guide is primarily used by administrators, but also by developers.

Purpose and Description

The Application Deployment Guide describes how to verify and deploy modules and applications, including instructions on using the deployment descriptors specific to the GlassFish Enterprise Server. Assembly is not addressed.

Statement of Work

There are three phases in the GlassFish v3 project: Prelude (June 2008), Preview (June 2009), and FCS (September 2009).

TP2/Prelude Statement of Work

(Writer: June Parks) For Prelude, the Application Deployment Guide will include only information on developing web modules. The verifier utility will not be not included.
Table of contents:

  • Chapter 1 Assembling and Deploying Applications
  • Appendix A Deployment Descriptor Files

Java ONE Preview Statement of Work

For JavaONE Preview, content for applications and other module types will be restored. New writer (June handoff to Dixie) will reorganize the document in a task-based model. All asadmin commands associated with application deployment will be moved from the Administration Guide to this guide. The verifier utility will not be included.
Table of contents:

  • Chapter 1 Overview of Application Deployment
  • Chapter 2 Deploying Applications
  • Appendix A The asadmin Application Deployment Commands
  • Appendix B Enterprise Server Deployment Descriptor Files
  • Appendix C Elements of the nterprise Server Deployment Descriptors

FCS Statement of Work

For FCS, application start/stop, changing of servlet init parameters, and changing of env variables will be included. The verifier utility is in question.
Table of contents:

  • Chapter 1 Overview of Application Deployment
  • ADD Chapter 2 Verifying Applications
  • Chapter 3 Deploying Applications
  • Appendix A The asadmin Application Deployment Commands
  • Appendix B Enterprise Server Deployment Descriptor Files
  • Appendix C Elements of the nterprise Server Deployment Descriptors
