Audience

This feature is aimed at system administrators and application developers.

Summary of the Feature From a User's Perspective

This project defines GlassFish 3.1 product packaging based on the Image Packaging System (IPS). IPS packaging will make it possible to deliver the GlassFish Open Source Edition 3.1 release as a standalone product distribution through the Update Center 2.x content delivery mechanism. Product packages will be organized based on functional component boundaries and will be mapped to OSGi bundles. Package naming and versioning will follow IPS Packaging Best Practices.

With the exception of the glassfish-scripting package, all packages from GlassFish version 3.0.1 will be brought forward to GlassFish version 3.1. In addition, six new packages will be added to the GlassFish 3.1 distribution. These packages are described in the GlassFish v3.1 Packaging One-Pager.

Availability

This feature is available in all GlassFish Server distributions.
