  • Are you an existing Rails developer and like to know a simple, easy-to-use, and highly robust deployment environment ?
  • Are you a Java developer and interested in learning how Rails applications can be deployed in your environment ?
  • Would you like to learn how to leverage the maturity of Java platform with the agility of Ruby-on-Rails ?
  • Do you know JRuby, GlassFish, and NetBeans provide a complete development/deployment platform for Ruby-on-Rails ?

If you answered "yes" to any of the questions above, then you need to register for Community One West Deep Dive.

Time: 9am - 5pm
Lunch: 12pm - 1pm

Agenda

  1. Introduction to JRuby, Ruby-on-Rails, GlassFish, NetBeans (60 mins)
    1. Ensure environment setup
    2. Show Java and Ruby integration
  2. Hello Rails World - Develop, Run, Debug (30 mins)
  3. Merb, Sinatra, and Rack-based frameworks (60 mins)
  4. Know your GlassFish gem (30 mins)
  5. GlassFish Deployment Scenarios (90 mins)
    1. GlassFish Gem
    2. Directory-based
    3. WAR-based
  6. Warbler tricks (30 mins)
  7. Java EE and Rails integration (60 mins)
  8. Monitoring, Performance Tuning (60 mins)
