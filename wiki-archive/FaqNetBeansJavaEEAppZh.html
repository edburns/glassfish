能否使用 NetBeans 构建 Java EE 5 应用程序？

简单回答是可以。
有两个版本的 NetBeans 已支持开发 Java EE 5 模块和应用程序：请参见有关使用 NetBeans 6.0 开发 Web 组件以及其他 Java EE 5 工件的信息。另请参见有关使用 NetBeans 5.5 创建 Web 和 Java EE 5 工件的信息。
