There are three broad categories of changes happening to support SIP Application development for Sailfin:

  1. SIP Application Project module. This module allows users to create projects that create SIP applications. Items related to this module will be tagged with {PM}
  2. SIP Protocol Test Agent module. This module allows the user to test their applications at the protocol level. Items related to this module will be tagged with {TAM}
  3. Server Integration module changes. The GF integration plugin will be extended to expose Sailfin and its features, in the context of the application development environment that is NetBeans 6.0... Items related to these changes will be tagged with {SIM}
