What is this page all about

  1. Java Petstore 2.0 Early Access is a popular application to learn Java EE 5 technologies on GlassFish.
  2. Some people want to run Petstore 2.0 on Tomcat
  3. Tomcat needs a few jar files for extra technologies, such as JSF 1.2 and JPA, to run Petstore 2.0
  4. GlassFish contains separate jar files of various Java EE 5 technologies, such as JSF 1.2 and Java Persistence APIs, that can be used outside of GlassFish on other servers. For example, you can add JSF 1.2 and JPA jars from GlassFsih project to your Tomcat environment and play with the latest technologies.
