可于何处获取更为复杂的 Java EE 示例？

由 users@glassfish 邮件列表改述：

*问：*我是一名优秀的 Java 开发者，但我对 J2EE 和 GlassFish 不是很熟悉。我一直以来都在查看示例并学完了所有教程，但我找不到更复杂一些的示例。如何在复杂数据处理情形中充分利用有状态 bean？我对用户会话以及如何使用有状态 bean 与每个 bean 中的一个用户进行交互尤其感兴趣。

*答：*您可以查看 GlassFish Samples（GlassFish 样例）页面，其中提供了一些关于 EJB 3.0 的样例。所有这些样例都属于 NetBeans 项目，并可用于 NetBeans 5.5 或更高版本。请参阅样例文档来了解用法或任何其他问题。另外，还可查看 Java EE blueprint 应用程序。
