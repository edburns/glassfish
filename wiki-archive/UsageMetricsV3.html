GlassFish Usage Metrics

Starting with GlassFish V3 release, Sun may collect certain data transmitted by the bundled Updatecenter client as part of the download, installation, and auto-update processes (for full details, see below). This page provides specific information on the uses and types of data transmitted.

The collected data provides important feedback to Sun in order to improve the GlassFish and Updatecenter experience and optimize the programs and services we offer customers. If you wish to disable the collection and transmission of this data, you may choose to opt out at the time of installation. To disable the auto-update and collection of data at a later time using Updatecenter client configuration UI, please go to "Preferences" button, select "Updates" tab and set frequency to "Never".

WHO Sends Information to Sun?

Information under this program is collected and sent for all supported operating systems. It may be sent for any customers who download and/or install GlassFish V3 and do not opt out during installation. If the customer does not have an Internet connection while running GlassFish or Updatecenter client, installation data will not be returned to Sun.

HOW Is Information Sent to Sun?

Information is sent as part of HTTP request used to download Updatecenter catalog file or modules.

WHEN Is Information Sent to Sun?

Information is sent in following situations:

  1. When user starts Updatecenter client, it checks for available software and updates.
  2. Whenever user chooses to download and install available software or updates.
  3. Check for available updates will be performed when GlassFish domain is started according to configured check schedule.
  4. Check for available updates will be performed when user logs into GlassFish Admin GUI according to configured check schedule.
  5. Check for available updates will be performed according to configured check schedule if Updatecenter client is running as desktop tray icon process.

WHAT Information is Sent to Sun?

Some or all of the following information may be sent to Sun depending on the type of transaction:

Data Description
Installation ID Unique installation identifier GUID.
IP address IP address of the client so the country code lookup can be performed.
OS information Name, version, architecture, locale.
JRE version Java Runtime Environment version.
Download information Module name, date, time, status, download time, number of bytes downloaded.

For more information or if you have further questions about this program, please contact <a href="mailto:glassfish-data@sun.com">glassfish-data@sun.com</a>
