3.1 SSH AS Arch Review: setup-ssh and install-node

Oct 6, 2010

This review is for two new commands that were introduced in MS5. These commands are both convenience utilities and their use is optional when setting up systems for use in a GlassFish cluster that uses SSH nodes.

Sepcifications Description
setup-ssh A convenience utility to generate (if needed) an SSH key and propagate it to a set of systems to prepare them for use as GlassFish SSH nodes. Does not alter the DAS configuration.
install-node / uninstall-node Convenience utilities to install (uninstall) GlassFish on a set of systems using the DAS installation as the source. Does not alter the DAS configuration, but does query the DAS for some basic information.
