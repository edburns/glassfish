Introduction

This is the overall master task list for all the tasks we need to do. I would request all the admin developers to take a good look at this list and enhance it. Too much of information (especially if it is not coordinated) leads to confusion. So, I have tried to be precise here. This list should be contacted by everyone involved in administration for GlassFish V3, only because I think it is more convenient to do so. I know this is not WYSIWYG, but I couldn't help it. I encourage everyone to make this document more and more dependable and accurate. This should reflect the state of various projects/tasks in the administration area at a given point in time. I would also request everyone to be as precise as possible when defining the tasks. Note that these tasks are also to be linked to from the sub-projects page.

Ideally, an entry in this table should be one line wide. The focus of this list is on TP-2 release, but it may well go beyond that. The document that describes a task in sufficient detail should be a Wiki page or a plain HTML (no fluff) and should give all the details for a specific task.
