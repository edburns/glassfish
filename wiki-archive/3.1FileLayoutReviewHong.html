hong-1 4.5.1.1 GlassFish v3 File Layout I noticed the nodeagents directory is in the file layout, does that come with the default installation? Do we have two separate installations in 3.1, one for PE and one for EE or this is shared by both? We will not have separate PE/EE bundles. Clustering support will be included in all initial product distribution bundles. Note that zip bundles will continue having only preconfigured default DAS out of the box and users will have to configure the rest after the archive is extracted if they want to run a cluster or standalone instance. Installer bundles will provide options to configure DAS, instances, or standalone MQ broker on the system at installation time - you can check installer one-pager for more details on that.
