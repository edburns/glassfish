GlassFish v3 TP2 に何が入っていますか?

GlassFish v3 Technology Preview 2 (TP2)
は、現在開発中の GlassFish v3 アプリケーションサーバーをいち早く皆様に見て頂けます。

TP2 にはアプリケーションサーバーの機能や、Java EE 5 プラットフォームの技術のサブセットが入っているので、すべてがそろっているというわけではありません。現在入っているのは核となる v3 モジュール、Java EE ウェブ層の技術、Java Persistence API そして非 Java プログラミング言語のコンテナです。
TP2 の内容リスト(英語) に、TP2 に入っているものが記述されています。
さらなる情報は TP2 リリースノート(英語） もご覧ください。
