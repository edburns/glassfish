GlassFish v3 Error Message Reference

The GlassFish v3 Error Message Reference describes error messages that you might encounter when using GlassFish v3 (Sun GlassFish Enterprise Server v3).

Error messages provide information about events that occur during GlassFish v3 operation, such as configuration errors, security failures, or server malfunction. These messages often include diagnostic and troubleshooting information.

The GlassFish v3 Error Message Reference is also available in HTML and PDF formats. This wiki version is intended to be an interactive, editable version of the guide. It is also intended to be an expanded version. You and other members of the GlassFish community are asked to add information and make this resource as complete and useful as possible. More about that in How to Contribute.

Error messages can also be viewed in the server.log file and in the Administration Console Log Viewer. For more information, see
Viewing Log Information in the ''[Sun GlassFish Enterprise Server v3 Administration
Guide|http://docs.sun.com/doc/820-7692]. For additional troubleshooting information, see the[Sun GlassFish Enterprise Server v3 Troubleshooting
Guide|http://docs.sun.com/doc/820-7699]''.

For complete GlassFish v3 documentation, see the GlassFish v3 documentation collection.

How This Guide is Organized

Error messages are listed in alphabetic and numeric order by message ID. The text is as it appears in the actual error message. Each message ID includes a prefix that represents a subsystem within the software. Error messages for the following prefixes are provided:

  • ACC
  • ACDEPL
  • ACT
  • ADM
  • CMNUTL
  • CORE
  • DIAG
  • DPL
  • DTX

  • WS

How to Contribute

Error messages in this guide include Cause and Solution sections, which describe possible causes of a problem and possible solutions.

You'll also see many entries with TBD, which indicates that the Cause and Solution information for a particular error message has not yet been provided in the GlassFish code, and thus is not available in this guide (this guide is derived from what's in the code). If you have that information, please add it to the wiki page! The fix should also be made (information added) in the code.

Please also enhance this guide in other ways, such as adding error messages that aren't listed, adding new wiki pages for prefixes that aren't included, and so on. This is meant to be a dynamic, community-driven resource, revised and updated by all for the greater good. For more information about editing these pages and adding content, see How do I edit a wiki page? Contributions are governed by the GlassFish wiki terms of use.

Questions about any of this? Contact the documentation team at [docs@glassfish.java.net]. You can also post a comment on the Error Message Reference comments page.
