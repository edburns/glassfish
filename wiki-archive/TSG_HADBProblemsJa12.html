実行不可能なクラスター設定

  • 解説
    \二つの異なる HADB のインストールが設定されます。 : ひとつはサーバーホスト上で、もうひとつは hadbm ホスト上でそれぞれ異なる バージョンの HADB を実行しています。管理エージェントは、 HADB バージョンのうちのひとつで起動され、 hadbm create が、もうひとつのバージョンによって、実行されます。次のエラーが返却されます。 :\ 
    HADBMGMT007:hadbm create command failed.  Return value: 1 STDOUT:
STDERR: hadbm:Error 22170: The software package V4.4.x could not be
found at path <pacakgepath\>/4.4-x on host <hostname\>.
CLI137 Command configure-ha-cluster failed.
  • 解決方法
    \管理エージェントと、全ての hadbm クライアントに対して、同じバージョンの HADB を使用してください。
