将 Apache 用作 GlassFish 前面的负载平衡器

  • 此讨论可以作为起点。
  • Prashanth 的博客提供了在 GlassFish V2 中进行负载平衡的精彩概述。
  • Jan Luehe 的博客介绍了如何配置 httpd 和 GlassFish；但其中还包括体系结构概述，说明 jvmRoute 在 httpd 所使用的负载平衡器架构中的工作原理以及实现方式。

在对以下问题的回答中：
要获得负载平衡，GF 有用于 Apache Web 服务器的插件吗？

Shreedhar.Ganapathy 回答：
Sun Java System Application Server 9.1 正式支持使用 Apache Web 服务器和 mod_jk 插件进行负载平衡，还支持使用 Microsoft IIS 6 以及 Sun 的 Web Server 6.1sp5 和 7.0。blogs.oracle.com 或 java.net 上有很多关于支持使用 Apache Web 服务器对 GF 进行负载平衡和代理的博客文章。以下就是这样一篇博客文章，其中含有一些附加链接：http://blogs.oracle.com/dadelhardt/entry/loadbalancing_with_mod_jk_and_glassfish
