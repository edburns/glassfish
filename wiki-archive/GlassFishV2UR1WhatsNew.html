What's New in GlassFish v2 Update 1 Release

Scope of Release (Goals)

GlassFish v2 Update 1 has been released on December 19, 2007. The final GlassFish v2 build for Update 1 release is b09d.

This is a tightly constrained release to accommodate high priority bug fixes. The following are the highlights of the release:

  • Added AIX Platform support for Cluster and Developer Profiles - versions supported 5.2 and 5.3
  • Added 64 bit JVM support for Suse Linux 64 bit OS.
  • Specifically targeted bugs & enhancements that are a "must have"
  • OpenESB Requirements
  • Late Customer Feedback from GFv2 beta release
  • Netbeans critical bugs
  • Community requested feature for enabling sticky session support for load balancing using Apache with mod_jk.
  • Added new user Registration UI to Administration GUI console and to Update Center to enable customers to register the product. In the Sun branded SJSAS version, the installer also contains this registration UI. The registration is a one-time action.
