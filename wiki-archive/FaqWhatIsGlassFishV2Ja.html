GlassFish v2 とは何ですか?

GlassFish v2 は GlassFish コミュニティが開発しているオープンソースの Java EE 5 アプリケーションサーバーです。
GlassFish v1 は最初に利用可能になった Java EE 5 アプリケーションサーバーで開発者向けの機能にフォーカスしていました。後継の GlassFish v2 は、エンタープライズ向け機能(英語)を追加しています。

GlassFish v2 は 2007年9月17日にリリースされ、こちらからダウンロードできます。
GlassFish v2 にサンのサポートをつけたものが、Sun Java System Application Server 9.1 で、同様に 2007年9月17日にリリースされました。GlassFish v2 と Sun Java System Application Server 9.1 は同じ機能を提供しています。
