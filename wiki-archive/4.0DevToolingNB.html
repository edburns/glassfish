Introduction

Goal / Scope

The goal of this project is to allow users to develop Java EE 6, Java EE 5 and J2EE 1.4 based apps and deploy them onto GlassFish Server 3.2 as easily as possible from inside the most recently released version of the NetBeans IDE available.

Since the NetBeans roadmap beyond 7.0, which will ship in April 2011, is unclear; there may be some work necessary to create an update to 7.0 that includes the ability to interact with GlassFish Server 3.2... This update would be on a different 'track' than the main body of the NetBeans codebase.
