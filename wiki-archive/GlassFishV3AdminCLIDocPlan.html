Summary of the Feature From a User's Perspective

This feature affects how users will administer GlassFish Application Server from the command line.

Availability

This feature is available all distributions of GlassFish Application Server.

Command-Line Utility for Administration (asadmin) . Moderate Paul Kedar Mhasawade
Lloyd Chambers
Bill Shannon
Sankar Neelakandan 		Add these subsections:
  • Format ofasadmin Commands
    Explain that the general format of asadmin commands is: asadmin asadmin-utility-options subcommand subcommand-options operands
    Cross-refer to the asadmin(1M) man page for the full syntax.
  • Subcommands of theasadminUtility
    Explain that subcommand specifies the operation or task that the user is performing.
    Cross-refer to the Reference Manual for the full list of asadmin utility subcommands.
  • Options of theasadminUtility
    Explain that:
    • These options control the behavior of the asadmin utility, not the subcommand.
    • These options must precede the subcommand.
    • For compatibility with earlier releases, these options are permitted after the subcommand, but such syntax is deprecated.
  • Options of asadmin Subcommands
    Explan that:
    • Subcommand options control the behavior of the the subcommand, not the asadmin utility.
    • Subcommand options must follow the subcommand.
    • A subcommand option may have the same name as an asadmin utility option, but the effect of the two options are different.
  • Operands of asadmin Subcommands
    Explain that:
    • Operands specify the objects on which the subcommand is to act.
    • Operands must follow the subcommand options.
