カスタム JSP タグの実装

本バージョンでサポートされる独自のカスタム JSP タグに関する制約を以下に示します：

WAS 4.0 用カスタム JSP タグ

<tsx:getProperty> タグを使ってBean を操作することはできません。

WLS 5.1 用カスタム JSP タグ

Application Server は <wl:cache> タグを認識しますが、このタグは no-op （訳者注： No Operation / 何もしないこと） であると見なされます。このタグは WLS 5.1 上ではキャッシュ機能を提供しますが、 Application Server は当該機能をサポートしていません。
このタグを含む JSP のマイグレーションの失敗を回避するため、タグは no-op として実装されます。

WLS 6.x 用カスタムタグ

<wl:prefetch> タグおよび <wl:beanparam> タグはサポートされません。 また、<wl:cache> タグの実装は、 "クラスタ" スコープおよびキャッシュ管理機能をサポートしません。
