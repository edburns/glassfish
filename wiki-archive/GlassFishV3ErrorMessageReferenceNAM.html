----NAM0004 Exception during name lookup : {0}

Cause

App Server may not be running at port intended.

Cause

Possible Network Error.

Solution

Check to see if the AppServer is up and running on the port intended. The problem could be because of incorrect port.

Solution

Check to see if you can access the host on which the AppServer running.

----NAM0005 JMS Connection Factory resource not found: {0}

Cause

The JMS Connection Factory may not be bound to a JNDI name.

Solution

Check the list of JNDI bindings by clicking on the JNDI tree on the Admnistrator's console. Check the JMS resource references bindings.

----NAM0006 JMS Destination object not found: {0}

Cause

The JMS destination may not be bound to a JNDI name.

Solution

Check the list of JNDI bindings by clicking on the JNDI tree on the Admnistrator's console. Check the JMS resource references bindings.
