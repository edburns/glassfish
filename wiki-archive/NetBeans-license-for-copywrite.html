NetBeans License File for GlassFish Copyright

I have had good luck with this sample NetBeans license file for GlassFish. It works when you create new files, follows the current (as of this writing) copyright requirements for GlassFish, and automatically inserts the current year in the right place. NetBeans can apply a single license template file to several different kinds of sources (Java, properties, etc.)

For my purposes I have simply edited the NetBeans default license text. (In NetBeans 6.9 and 6.9.1 - and, I think, recently-preceding versions - use Tools ->Templates, open the Licenses group and select Default License. Click Open in Editor. Replace what's there with the contents of the attachment and save and close that window.)

NetBeans seems to support the property project.license which would allow finer control over which new files get the GlassFish license vs. others, but I have not spent much time trying to get that to work; I think to do it right might involve adding a hint of some kind to our pom(s). Almost every file I create is for GlassFish and so should include the copyright, so I have not had much of a problem using the GlassFish copyright for every new file I create.
