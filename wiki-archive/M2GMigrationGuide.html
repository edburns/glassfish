Migration Guide for GlassFish

The purpose of this migration guide is to help in migrating applications to GlassFish.

How To Contribute

  • Login ("Log in" button at the top ) with a GFWiki id.
  • Create a new page for the migration issue and solution under the appropriate section.

JBoss -> GlassFish Migration Issues

  • Local EJB reference using a non standard JNDI name
  • Classes packaged at top level in a WAR file
  • Using Facelets
  • JBoss EJBQL != Glassfish(J2EE Spec) EJBQL
  • Embeddable objects referenced as Entities in Glassfish Cache
  • GlassFish cache refresh
  • Using User Transaction in a Bean using CMT
  • Context Root for war file name starting with number
  • Session Bean and Component Inheritance
  • taglib-location

WebSphere -> GlassFish Migration Issues

  • Invalid URL Pattern in <servlet-mapping> in web.xml
  • extendedDocumentRoot - sharing of tag files between web apps
  • Verification Tools for Migration To GlassFish

WebLogic -> GlassFish Migration Issues

  • Package import in JSP page
  • WebLogic's QL Extensions for EJB
  • Case Sensitivity of JSP tags/attributes
  • Sharing classes using APP-INF
  • printNulls (empty vs "null" strings)
  • Split Directory To ear file

Tomcat -> GlassFish Migration Issues

  • Custom Valve
  • Leading/trailing spaces in URL
  • java.lang.AbstractMethodError on JSP page access
  • Support for context.xml
  • Changes to defult-web.xml at runtime

JaxRPC -> JAX-WS Migration Issues

  • Fast Infoset Content Negotiation
  • Java EE Module Loading Order(I)
  • Verification Tools for Migrating to GlassFish
