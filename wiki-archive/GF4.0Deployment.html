Feature Overview

  • Java EE 7 platform spec (deployment related requirements)
    • Support external alternate deployment descriptors
    • Other clarifications (for example, annotation/descriptor overriding rules)
  • Deployment order
    • Specify a deployment order at application deployment time, and the applications will be loaded based on that order during server start up
