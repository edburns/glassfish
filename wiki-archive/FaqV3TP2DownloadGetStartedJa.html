GlassFish v3 TP2 を入手して使い始めるには?

GlassFish v3 Technology Preview 2 ページに行って、TP2 をダウンロードし、使い方が書かれた情報を参照してください。
このページには、
インストール手順(英語) と
クイックスタートガイド(英語)
がありますので、これらを見て頂ければすぐに使い始められると思います。
他の情報源としては、
追加の TP2 ドキュメント(英語) や TP2 向けの Java EE 5 チュートリアルのロードマップ(英語) や デモ(英語), screencasts
や 便利な情報やブログ(英語) や
ユーザー向け および
開発者向け(英語) FAQ があります。

問題やコメントや質問があれば、
GlassFish の [ユーザーメーリングリスト] か、あるいは
GlassFish フォーラム まで
お問い合わせください。
