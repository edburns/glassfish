Audience

This feature is aimed at system administrators, application developers, and product support engineers.

Summary of the Feature From a User's Perspective

The Logging & Diagnostic components provide GUI and command-line tools for configuring, searching, viewing, and archiving server logs. These features will be available for standalone instances, DAS, and cluster nodes.

Availability

This feature is available in all GlassFish Server distributions.
