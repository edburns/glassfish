Audience

This feature is aimed at system administrators and application developers.

Summary of the Feature From a User's Perspective

The Installer components provide GUI and command-line tools for performing GlassFish installations and upgrades. As part of these features, the Installer supports custom and standard installations, and installations in standalone or clustered modes, including DAS and individual nodes.

Availability

This feature is available in all GlassFish Server distributions.
