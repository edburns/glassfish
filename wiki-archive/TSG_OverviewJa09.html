ログ オプション

管理GUIは、以下の２つのログに関するオプションを提供しています:

  • Option 1: stdout (System.out.print) への出力を event log へ出力します。
  • Option 2: stderr (System.err.print) への出力を event log へ出力します。

これらのオプションが設定された場合、stdout と stderr のメッセージは server.log ファイルに出力されます。(event log は、Solaris の　syslog デーモンであり、Event Log は Microsoft Windows のものです。 )

上記のオプションが設定されていない場合:

  • stdout または stderr へ出力されるメッセージ(System.out または System.err を使用して出力されるメッセージ)は、ログファイルには出力されません。
  • JDK の logger を使用して出力されるメッセージは、ログファイルに出力されます。
  • stdout または stderr へ出力されるメッセージは INFO レベルですが、message ID を持たない形で出力されます。
