注 Java EE 5 SDK にバンドルされている Sun Java Application Server 9.1 は GlassFish v2 の商用バージョンです。2 つのコードペースは同じで、違いは提供されるサポートオプションとなります。このウィキとしては、「Application Server」は全て等しく「GlassFish」をさすものとします。つまり名称は相互に入れ替え可能です。Application Server 9.1 と Java EE 5 SDK に関する情報は、Java EE ダウンロードのページを参照下さい。

前書き

Sun Java System Application Server 9.1 トラブルシューティングガイド は Sun Java System Application Server 9.1 を使用する上で直面することがある共通の問題とその解決方法を説明しています。

対象読者

Sun Java System Application Server 9.1 トラブルシューティングガイド は Sun Java System Application Server を使用して web ベースの分散アプリケーションを構築、配備する開発者と管理者を対象としています。

本ドキュメントの構成

このウィキは Sun Java System Application Server 9.1 ソフトウェアを設定、利用、もしくはアプリケーションを配備する際に直面する可能性のある問題の回避策を解説します。以下の章を含みます。

  • 第 1 章, 概要 (原文 ) トラブルシューティングのプロセスについての一般的な概要。
  • 第 2 章, 共通の問題 (原文 ) 製品を使用する際に直面する可能性のある共通の問題。
  • 第 3 章, HADB の問題 (原文 ) HADB に関連する問題。
  • 第 4 章, セキュリティの問題 (原文 ) セキュリティの設定に関連する問題。
  • 第 5 章, FAQ (原文 ) Application Server ソフトウェアに関する FAQ。
注 ウィキが成熟するに従って、章が追加されることが予想されます。
