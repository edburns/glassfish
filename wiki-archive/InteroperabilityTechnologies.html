GlassFish Extras - Interoperability Technologies

Technologies that enable interoperability between GlassFish and Other Middleware

We encourage you to contribute any additional interoperable technology to this list.

[PostgreSQL http://www.postgresql.org Pramod Gopinath writes about how to configure PostgreSQL for using in <nop>GlassFish Tested on Build 38.
WebSphereMQ Siva Thyagarajan shows how to integrate WebSphereMQ with SJSAS/GlassFish using the [ GenericJMSRA Adapter https://genericjmsra.java.net]. Test build : unknown
TIBCO EMS Narayanan.V, Sivakumar Thyagarajan, and Binod PG have written an article on Integrating Sun Java System Application Server with Tibco EMS using the Generic Resource Adapter for JMS Test Build : Unknown
SoniqMQ In an article, Ramesh Parthasarathy shows steps to Integrate Sun Java System Application Server with Sonic MQ 7.0 using Generic Resource Adapter for JMS Test Build: GlassFish FCS/Java EE 5 SDK
JAX-WS Arun Gupta blogs about the 100% interoperability achieved with JAX-WS 2.1 (Web Services Addressing) using Sun's implementation as part of W3C WS-Addressing interoperability testing effort.  
Mustang Mustang works with GlassFish    
Apache httpd as Proxy Jeanfrancois Arcand has posted detailed steps to configure Apache HTTP server to proxy requests to GlassFish using mod_jk. Fixed in build 43, support mod_jk and mod_proxy_jk.  
Using Apache mod_jk as LoadBalancer Daniel shows how to load balance GlassFish instances with Apache's mod_jk library Tested with UR1  
Sun Secure Application Switch Clustering and Securing Web Applications: A Tutorial explains how multiple GlassFish instances can be used with a loadbalancer to scale horizontally. Tested on Build 38
Derby Database Derby is bundled with <nop>GlassFish. Lance explains how to start/stop the database. Tested on Build 38
JUEL How to use JUEL as an alternative EL implementation with GlassFish V2 Tested on V2 M2 (build 19)
ActiveMQ Active MQ has been tested to work with GlassFish as per the ActiveMQ home page. Tested Build : unknown
