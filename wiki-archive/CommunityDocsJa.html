GlassFish コミュニティドキュメント

GlassFish ドキュメントコミュニティ向けの、実用的な GlassFish プロジェクト向けコミュニティドキュメントのページへようこそ。

そのほとんどは、 GlassFish ドキュメントホームページの一覧に記載されていて、 GlassFish ドキュメントの貢献方法が数多く存在します。 ドキュメントレビューを含む、テスティングプログラムと言う新しいコミュニティ、 FishCAT を、必ず見てください。

この、コミュニティドキュメントページは、コミュニティドキュメントで、如何に貢献するかの題材、例えばチュートリアル、wiki ページの How-To やアーティクル、スクリーンキャスト、 FAQ 、そして同じようなドキュメント等にフォーカスしています。又、このページには、コミュニティの貢献についても、記録されています。そしてそのページは、ドキュメントコミュニティが成長し、ドキュメントコミュニティ活動が、機運を得ることによって、成長します。それら一連の行為は、作業の進捗と、(ヒント ヒント)にウェイトを置かれたコミュニティとして、洗練されます。

質問、コメント、又は、提案するには ? ドキュメントメーリングリストに連絡して下さい。 (そしてもちろん、もし未だあなたが参加していなければ、 GlassFish メーリングリスト ページから、メーリングリストに参加してください)。

"公式な" 製品ドキュメントを見つけるには ? 最新の GlassFish v3 マイルストーンリリース、 GlassFish v3 テクノロジープレビュー 2 (TP2) については、 TP2 ドキュメント ページを参照ください。GlassFish v2 (Sun Java System Application Server 9.1 and 9.1 Updates) については、 GlassFish v2 ドキュメント サイトを参照ください。また、 GlassFish ドキュンメントを説明している FAQ も、参照ください。

ドキュメントメーリングリストへの参加

あなたが java.net アカウントをお持ちか確認し、以下の手順を踏んでください。アカウント取得方法についての情報も又、記載しています。
ドキュメントメーリングリストへ参加するためには

  1. GlassFish メーリングリスト ページにて Login をクリックします。
  2. ログインページにて、 java.net に登録したあなたのユーザー名又は、メールアドレスと、パスワードでログインします。
    もしあなたが登録していない場合は、次のリンクでアカウント登録し、メールで受け取ったパスワードでログインしてください。
  3. ログイン後、 docs@glassfish.java.net を見つけられるまでページを下の方へスクロールしてください。
  4. ご希望されるフォーマットを選択してください。: メッセージ毎又は、ダイジェスト.
  5. 購読をクリックしてください。
  6. あなたが受信した、購読申し込み確認メールに返信して下さい。

これで、 GlassFish ドキュメントコミュニティでの議論に、参加する準備が整いました。

コミュニティドキュメントへの貢献方法

貢献する為の方法に関する、アーティクル、チュートリアル、ヒントと秘訣、スクリーンキャストと同様のドキュメント

  1. あなた自信がドキュメント化する、あるいは、確認したトピックに必要性があるものを見つける為に、コミュニティドキュメントウィッシュリストをチェックして下さい。
  2. あなたが何に貢献するかを意図しているかを記載したメールを、[ドキュメントメーリングリスト]へ送信してください。
  3. コミュニティドキュメント作業進捗表に、あなたが作業しているドキュメントを記録してください。
  4. 作業が完了したら、コミュニティドキュメント貢献者一覧 表に、あなたの貢献を記録してください。又、 How-To 集向けの、GlassFish How-To's ページに、リンクを追加してください。

注記 : How-To の書き方についても、ご一読してください。
