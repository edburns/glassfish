List of Commands

  • CLI commands are listed below, for the syntax please refer to link

Application Management:

  • deploy/undeploy (no java2db)
  • enable/disable
  • list-components
  • redeploy
  • deploydir (deprecated. use deploy command instead)
  • show-component-status (Nice To Have, here after 'NTH')

Infrastructure:

  • backup/restore-domain
  • change-admin-password (NTH)
  • create/delete-ssl
  • create/delete/list-auth-realm
  • create/delete/list-domain
  • create/delete/list-file-user
    • create/delete/list-mbean-
      * get/set/list
      * help
  • start-domain/stop-domain
  • update-file-user (RSR)
  • version

* create/delete/list-jvm-options (RSR)

    • multimode (NTH)-

Runtime Management:

  • create/delete/list-jdbc-connection-pool
  • create/delete/list-jdbc-resource
  • create/delete/list-http-listener (RSR)
  • create/delete/list-virtual-server
  • ping-connection-pool
  • create/delete-profiler
  • add-resources
  • start-database/stop-database
  • create/delete/list-system-properties (NTH)
    • create/delete/list-javamail-resource (NTH)-
    • monitor (NTH)-

Current list of working commands is here.

Note

RSR implies Requires Restart for TP2. The goal is to have most of the things reconfigured dynamically without restart.
NTH implies Nice to Have.
