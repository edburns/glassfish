Welcome to GlassFish v3 EA Community Localization Project!

Introduction

Glassfish v3 is scheduled to be released in Sept. 2009 (refer to v3 schedule). To help community to get an early access of this product in their own languages, we plan this project to give Glassfish users an opportunity to contribute their translation and testing. The project will start in early May and end by the end of June 2009. We have designed an online translation tool called (CTI) which leverages our past GlassFish translations (huge amount!). We also created test cases and give bug fixing information to assist testing. We welcome community members to participate in this project and make your contribution to
Glassfish v3 program.

Try Early Access (EA) L10n Bundle

Localization zip bundles are available here. They are built weekly. For install instructions and how you can use it, please refer to install instruction page. We always appreciate your feedback. Please send them to g11n AT glassfish.java.net.

Project Details

In this EA Community Localization Project, willing contributor can take part in translation and/or testing.

Translation

In previous v3 Prelude ML release, GUI/CLI localization were done into 7 languages listed below.

de es fr ja ko zh_CN zh_TW

We look forward to work with interested community to add new languages here. And, yes, we still need help for above 7. There are updated and new messages.

For details on how to work on translation, please refer to translation steps.

Testing

Please try out our l10n bundles and send us your feedback. L10n is also participating in Quality Community so people there are also very welcome.

For details, please refer to

  • EA L10n testing instruction page (EA specific information) and
  • L10n Testing page (for generic l10n testing).

Schedule

Starting 5/1, people can take part in both translation and testing. L10n zip bundle is built weekly until the end of this EA project. After the EA project ends, we will move to FCS phase.
For details, please refer to schedule page.

Rewards

Active contributors will be rewarded prizes. We value translation volumes, numbers and impact of the bugs reported, and passionate feedback. Prizes TBD:

  • 2 PSPs
  • 1 Digital Photo Frame
  • 1 GlassFish iPod shuffle
  • JavaOne GlassFish T-shirts (30 max.)

Winners will be announced within 2 weeks after the program ends. Please check if your name is listed on contributors page contributors page and let us know if not so we don't miss your entry.

Communication

We will use g11n AT glassfish.java.net for all communications. So anyone join the project, please subscribe.
Any questions/feedback, please send them here.
