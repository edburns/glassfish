GlassFish v3 とは何ですか?

GlassFish v3 は、次期メジャーリリースの GlassFish アプリケーションサーバーで、現在 GlassFish コミュニティによって開発が進められています。
GlassFish v3 はモジュール化された、ライトウェイトで、拡張可能なアーキテクチャになっています。次に上げられたものを提供するのが目標です:

  • オープンで、モジュール化され、拡張可能なプラットフォーム
  • 速くて、使いやすく、信頼性のあるアプリケーションサーバー
  • スケーラビリティに優れ、洗練された管理機能を持った、エンタープライズ用途に使えるアプリケーションサーバー
  • 製品のアップデートとアドオンを、実務で使える Update Center 2.0 から提供
  • OSGi のサポート
  • Ruby や Groovy といった動的言語のサポート
  • Java EE 6 のサポート

概要

GlassFish v3 は、次期メジャーリリースの GlassFish アプリケーションサーバーです。このリリースでのフォーカスは、モジュール化と Java EE 以外のコンテナ、そして組み込み機能です。

GlassFish V3 technology preview をぜひお試しください! いくつかシンプルな例があります。
