This page contains the list of subcommands in GlassFish Server 4.0.

The list includes:

  • Subcommands that are part of the public user interface.
  • Subcommands that are internal and are not intended to be run directly by users.
    The name of an internal subcommand starts with an underscore (_).

For each subcommand, the following information is provided:

  • Subcommand. The name of the subcommand.
  • Mode. The mode of the subcommand: local or remote.
  • Path. The location of the subcommand beneath trunk/main/ for source code and beneath trunk/main-docs/man/ for the man page.
  • Package. The package in which the subcommand is delivered.
  • main-docs Module. The module in which the man page for the subcommand is delivered.
  • Release. The release when the command was supplied:
    • 3.1.2. The subcommand was supplied in the 3.1.2 release.
    • 4.0. The subcommand is new in the 4.0 release.
  • Notes. Any notes relating to the subcommand.

The value "TBR" means "to be researched"; a prefatory "OOD:" means "out of date", indicating that the following value is not current.
