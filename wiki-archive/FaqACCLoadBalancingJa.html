アプリケーションクライアントコンテナ (ACC) でロードバランスを利用できますか？

はい、 ACC と共にロードバランスを使うことができます。

Java Web Start を使ったアプリケーションクライアントの起動にビルトイン
サポートを使っている場合、何か特別なことをする必要はありません。
クライアントを起動したサーバーがクラスターの１つであれば、自動的に
ロードバランスします。

一方、クライアントの起動に appclient スクリプトを使っている場合、
[開発者ガイド（英語)

https://glassfish.java.net/nonav/javaee5/docs/AS91DG.pdf]の１１章をご覧ください。"To Access an EJB Component Froman Application Client."
のセクションです。現在のドラフトでは（常に修正する可能性があるので）、
番号をふったリストがあり、６と７にクライアント側でロードバランスを有効に
するための説明があります。

基本的に、１つの XML ファイルを編集するだけでサーバーの正しいホストと
ポートの値を参照します。

元の回答がポストされたのは
こちら
です。
