Audience

Java EE application developers are primary users of the Application Client Container (ACC) feature. Administrators who deploy client JAR files only need to know about the most basic ACC features.

Summary of the Feature From a User's Perspective

The new JNLP enhancements give advanced developer users more control over the existing Java Web Start feature of the ACC.

Availability

The ACC is available in the Full Platform distribution of GlassFish Server. It is not available in the Web Profile distribution.
