Summary of the Feature

Product installer/uninstaller for GlassFish V3 distribution.

Availability

This feature is part of all GlassFish v3 distributions.

Release Notes Changes

Section Doc Impact Writer Reviewers Description of Change
Compatibility issues Moderate - Sathyan, Snjezana Mention that this installation program is not compatible with earlier releases of GlassFish installation interfaces as different incompatible frameworks are used to develop them. State-files(silent installation) generated out of the earlier versions of application server product installers cannot be used with this program.
