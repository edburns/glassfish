What
The GlassFish Wiki was migrated on July 4th, 2010 from a JSPWiki infrastructure running at http://wiki.glassfish.java.net to a Confluence-based wiki on the shared http://wikis.sun.com infrastructure.

Limitations
This was done using a migration tool which has a number of limitations documented here.

Old content
Each page has a link to its older version to help manually fix what the tool could not automate. It is a good practice to remove those links once the page content has been validated or manually updated. Use the http://wiki.glassfish.java.net domain name for older content. Other domain names (such as wikis.glassfish.org) now have a top-level redirect to the new Wiki home.

Wiki Syntax
The syntax used on wikis.sun.com is documented here : http://wikis.sun.com/renderer/notationhelp.action and the Rich Text editor is quite usable.
