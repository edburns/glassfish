Glassfish v3 Community Localization Plan

Overview

GlassFish v3 is an open-sourced enterprise application server, targeted to be released in late May 2009 with early access (EA) version and in September 2009 in FCS version. We would like to have the program localized so that people in different countries can use Glassfish in their own language for their production. Community members are welcome to join the effort and contribute translation to Glassfish v3. Sun' G11n team will assist you in every aspect of the localization process.

Schedule

  • Glassfish v3 Schedule

What to work on

  • Translate modules in product - Admin GUI, CLI, OLH. A translation tool is provide with available files. See Project 1 on G11n Home page
  • Translate Docs outside product - web pages, wiki pages and other docs. See Project 2 and 3 on G11n Home page
  • Testing - See Project 4 on G11n Home page

Builds and Releases

  • G11n team to provide ML zip files or builds
  • EA: l10n zip files to be available
  • FCS: ML bits to be release from java.net download pate

Communcation

  • For community community, use [g11n community alias]
  • For language group, use language group specific email alias
