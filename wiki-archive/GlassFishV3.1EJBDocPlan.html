Summary of the Feature From a User's Perspective

Enterprise JavaBeans features for GlassFish Server Open Source Edition 3.1:

  • SFSB Checkpointing (parity with v2)
  • EJB Timer Clustering / Failover (parity with v2)
  • Full EJB 3.1 feature support in Embeddable EJB API (new)
  • Retain SFSB / EJB Timer state across redeployment (new)
  • User-specified thread pools for EJB timer and async callbacks (dropped)
  • Option to disable v2 vendor-specific JNDI names (new)

Availability

EJB 3.1 Lite features are available in the Web Profile distribution of GlassFish Server. Other EJB features are available only in the Full Platform distribution of GlassFish Server.
