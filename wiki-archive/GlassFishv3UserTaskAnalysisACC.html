Attendees

Tim Quinn, Hong Zhang, Ken Saks
Facilitator: June Parks

Scope

User tasks and goals associated with the ACC that should be described in the user documentation, especially the Developer's Guide.

Related Information

  • One-pager
  • Dev Guide Chapter (v2.1)

List of Tasks

  • JavaFX
    • Compiling JavaFX script into a Java class
    • Placing Java class compiled from JavaFX script in appclient jar
  • Embedded ACC (API will change from spec)
    • Invoke ACC API to activate your appclient jar file or
    • Invoke ACC API to activate a class in your app
    • Including ACC jar file in classpath
  • ORB endpoints in appclient command, specifying using new option
  • Splash screen
    • Include image in applicent JAR with ref from manifest file or
    • Reference image in file system on java or appclient command
  • Launching appclient using java command
  • Zipping appclient jar files for copying to another location
  • Starting an app client with non-default JVM options

Associated Concepts

  • appclient jar is now smaller with dependent files in separate jars
