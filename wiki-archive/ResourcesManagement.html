Cool Features

  • Rolling Upgrade of Resources similar to Applications.
  • Reconfig now sends one event per change instead of batching and sending a cumulative event.
  • Application Scoped Resources
  • CallFlow enhancenments.
  • Monitoring
  • Auditing which will be common to the entire product.
  • Download jdbc-drivers from external website (url provided by User).
  • Install (sync) jdbc-driver copied to domains/<domain>/lib to all the machines.
