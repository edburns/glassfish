Audience

The administration infrastructure is mostly hidden from the average user, but there are some user-visible administrative features. Advanced users may want to view or edit the domain.xml file, in which the administration infrastructure is stored.

Summary of the Feature From a User's Perspective

The average user will see the administration infrastructure mainly in the Administration Console and the asadmin commands, which are covered in the doc plans for the Administration Console online help and the man pages. Advanced users will see the infrastructure in the domain.xml file, which is covered in the Domain File Format Reference doc plan. Special features for administering the infrastructure itself are described in this plan.

Availability

The administration infrastructure is present in every distribution of the GlassFish server, but the embedded version does not include a user-editable domain.xml file.
