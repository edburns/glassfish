<h1>GlassFish v3 User Task Analyses: Application Management</h1>
Tasks related to servlet init parameters, JNDI env values, and start/stop

_(2/27/08 June Parks leader, Dixie Pine recorder; attendees: Tim, Hong, Jane, Anissa)

Reconfiguring an App Without Manually Redeploying (App mgmt) Post-TP2

  • Edit subset of deployment descriptor using Admin Console (no CLI)
    • Change env values (cancel or apply)
      • Example: Change home path
    • Change servlet init parameters (cancel or apply)
      • Example: test a servlet

*Global Deployment Tasks
*

Before app deployment:

  • Set up JDBC connection pool
  • Set up other resources
  • Set up environment
  • If needed, set up security role mapping
  • Set up virtual server(s)

While app is running:

  • Enable application TP2
  • Disable application TP2
  • Application management Post-TP2

After app undeployment:

  • Undo resources
  • Undo virtual server
