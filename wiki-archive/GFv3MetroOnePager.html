2. Project Summary

2.1. Project Description:

Metro Web Services Stack == JAX-WS RI + WSIT/Tango (i.e., WS-*)

This is an umbrella one-pager for Metro. See the the one-pagers listed here for details in the following areas:

  • Monitoring and Management
  • Declarative Tubeline Assembler
  • WS-MakeConnection Support
  • Durable Reliable Messaging
  • Configuring Reliable Messaging via WS Feature
  • WS-I RSP Compliance
  • Dynamic Reconfiguration (note: will not be part of FCS)
  • WS-Policy Support for JAX-WS 2.2
  • WSIT configuration for Maven projects
  • Metro plugin for Eclipse
  • Issued Token Caching and Sharing, SSO among services
  • WS-Trust Renewing and Cancellation protocols
  • Support for WS-Trust 1.4, WS-SecureConversation 1.4 and WS-SecurityPolicy 1.3
  • Support for Password Derived Keys
  • Support for Annotations @RolesAllowed, @PermitAll and @DenyAll on Servlet WebServices
  • JAX-WS RI 2.2
  • SOAP/TCP 1.1
  • Web Services Integration
