我打算使用 Glassfish v2 应用程序客户机容器 (Application Client Container, ACC)。在哪里可以找到更多关于这方面的信息？

该问题最初在此线索中提出：

"我打算使用 Glassfish v2 应用程序客户机容器。在哪里能找到 jar 文件的列表/链接，可以包括在我的类路径中，以便使用这些功能？"

Tim 做了答复：

人们通常采用的两种做法是：

1. 使用 appclient 命令，这将在应用程序客户机 JAR 中启动客户机
或
2. 通过 Java Web Start 内置自动支持来启动应用程序客户机

以上两种方法都会为您获取所需的类路径。文档中对这两种方法进行了非常详细的介绍。开发者指南中的 Developing Java Clients（开发 Java 客户机）一章是一个不错的起点。
