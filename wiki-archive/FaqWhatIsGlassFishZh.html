什么是 GlassFish 项目？

GlassFish 社区正在开创免费、开源、生产级品质的企业软件。主要的交付产品包括应用服务器（Java EE 5 参考实现）和 Java Persistence API 参考实现 (TopLink Essentials)。

成熟的代码库是 Sun 将 Sun Java System Application Server 8.x 贡献给该社区的结果。严格来讲，该社区由 Sun 领导，当前正在制定管理方法。Oracle 和 Ericsson 也是主要贡献者，分别贡献了 Toplink Essentials 和 SIP Servlet 技术。

由 GlassFish 提供的可立即用于企业的功能包括：

  • 群集
    • 内存会话复制，可调节应用程序部署以满足要求。可以动态增大（或缩小）群集以满足客户要求。
    • 高可用性在服务器发生故障时可提供持续的服务可用性。
  • 高级集中管理
    • 管理群集（和单个实例）生命周期。动态创建、启动、停止和删除群集及实例。
    • 集中式群集配置
    • 集中式应用程序生命周期管理。从中央位置部署/取消部署应用程序。
    • 集中式监视（此处）（此处）（此处），也可以通过 JMX 公开给第三方工具。
    • 调用流监视，用于详细说明事务在应用服务器中的哪个位置花费了时间。
  • 业界最佳的性能
  • Microsoft .NET 3.0 的 Web 服务互操作性
