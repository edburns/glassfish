GlassFish v2 Update 1 Release

This Release was Delivered on 12/19/2007

Please see the What's New page for release information

Scope of Release (Goals)

GlassFish v2 Update 1 is a tightly constrained release to accommodate high priority bug fixes. Module Leads will be evaluating existing and incoming issues to see if they should be considered for integration into the release. The scope of the release is as follows:
1ST

  • AIX support for Cluster and Developer Profiles
  • Netbeans critical bugs
  • Late Customer Feedback from beta release
  • Specifically targeted bugs & enhancements that are a "must have"
  • OpenESB Requirements
    2ND
  • Ubuntu package support for HardyHeron
    Issue Tacking
  • Current List of GlassFish Approved Issues (for Target Milestone 9.1peur1)
  • Potential List of GlassFish Issues that Target the v2 Update 1 Release (for Target Milestone 9.1peur1)
  • Current List of OpenESB approved Issues
  • Current List of WSIT approved Issues
  • Current List of Update Center approved Issues
    Findbugs Error Report
    Check in requirements
    Bug Scrub List
  • Bugs that need to be Evaluated
  • How to evaluate a bug so it is removed from the Bug Scrub List
