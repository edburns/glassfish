  • GlassFish v2 will have one binary that contains clustering capabilities.
  • Initially there may be 2 binaries to start with and will be merged soon after. Goal is to merge only is the difference in size is not more than 10 MB.
  • By default, clustering features will be disabled
  • You will be able to select non-clustering or clustering mode when you install. (If possible, after installation also)
  • It will contain everything in Sun's EE version except for the HADB. A light weight in-memory replication may be included (based on when it comes in)
  • Source for all this will be available on GF cvs
  • There will be documentation and Load Balancer plugin (that can be downloaded separately) to configure a load balancer.
  • There will be documentation to install, configure node-agents, instances, clusters, etc.
