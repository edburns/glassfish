セマフォリソース不足が原因となる接続の問題

  • 解説
    \この問題は、履歴ファイルに次のメッセージが出力されると同時に、発生します。 :\\\\\~UWC_TOKEN_START~1278579878017~UWC_TOKEN_END
    \ホストコンピュータ上に、より多くのセマフォを構成するよう設定する必要があります。オペレーティングシステム上の semmnu の設定方法に関する情報について、 Sun Java System Application Server 9.1 インストールガイドの、 " HADB 設定準備 " の章を参照して下さい。
  • 解決方法
    \Stop the 影響を受けた HADB ノードを停止し、影響を受けたホストの再設定及び、再起動し、 HADB ノードを再起動してください。
