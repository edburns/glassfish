可于何处获得我提出的有关持久性和 Java Persistence API 的问题的答案？

Java Persistence API FAQ 是个很好的起点。

您可以从"规范"页面下载 JSR 220 (Enterprise JavaBeansTM 3.0) 的 Java Persistence API 部分。

《The Java EE 5 Tutorial》（《Java EE 5 教程》）的以下几章介绍 Java Persistence API 的用法：

  • Introduction to the Java Persistence API（Java Persistence API 介绍）
  • Persistence in the Web Tier（Web 层中的持久性）
  • Persistence in the EJB Tier（EJB 层中的持久性）
  • The Java Persistence Query Language（Java Persistence 查询语言）

还有一个有关 Glassfish 持久性的用户邮件列表，它也是以论坛的形式存在的，位于 Nable

Glassfish 持久性是 TopLink Essentials 产品，也可以将问题发布到 Oracle TopLink 论坛或 GlassFish 论坛。
