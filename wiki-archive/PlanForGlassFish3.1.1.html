Release Scope / Goals

Minor update release to the just released GlassFish Server Open Source Edition 3.1 version. This release is currently under development and will focus on the following:

  • Provide support for additional platforms including IBM's AIX.
  • Extend run-time environment support to include upcoming major and update releases of Oracle's Java SE implementations.
  • Critical bug fixes.
