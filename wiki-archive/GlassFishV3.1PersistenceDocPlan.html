Audience

Java Persistence is used by developers of applications that require database data to persist beyond single sessions.

Summary of the Feature From a User's Perspective

Persistence enhancements for GlassFish Server Open Source Edition 3.1 include:

  • Upgrade the EclipseLink persistence provider to v2.1, which has advanced ORM/query features
  • Provide a default data source for Embedded GlassFish Server

Availability

Java Persistence support is available in both the web and full GlassFish Server distributions.
