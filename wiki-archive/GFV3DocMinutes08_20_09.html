    • GlassFish v3 schedule has been changed and published. Put back changes by COB Friday Aug 21 for any man pages you want included in the Aug 24 Soft Code Freeze (SCF) build.

If a value is removed from domain.xml and the system restarted, the system reverts to the system defaults. June will list values in domain.xml as the defaults but in the description indicate that the system default is different.
