Introduction

JavaServer Faces technology simplifies building user interfaces for JavaServer applications. Developers can build web applications by assembling reuseable UI components in a page; connecting these components to ïœ­an application data source; and wiring client-generated events to server-side event handlers. This project provides information on the continued development of the JavaServer Faces specification. More information on JSF can be found at http://javaserverfaces.org/.

This page hosts status information taken from the official page of the JSF spec effort, https://javaserverfaces-spec-public.java.net/, and the official page of Oracle's Mojarra implementation project, https://javaserverfaces.java.net/. The information on this page is culled from those pages, and therefore may be stale.
