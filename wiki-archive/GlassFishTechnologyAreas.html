GlassFish Technology Areas

GlassFish covers the following technologies :

  • Java Programming
  • XML Technologies (JAXB, JAXP, StAX, FI))
  • Web Services (JAX-WS, WSIT, JAX-RPC, SAAJ)
  • Web Tier (Serlvet, JSP, JSTL, JSF)
  • Open ESB: Next generation platform for Service Oriented Architecture and Integration solutions
  • Persistence (Java Persistence API, TopLink Essentials)
  • Connectors (Connector Runtime, JDBC Resource Adapter, MQ Integration)
  • Databases (Java DB, MySQL , PostgreSQL )
  • SIP (Sip Container and Sip Servlet API - JSR 289)
  • Scripting (JavaScript, Phobos, JRuby, PHP, etc.)
