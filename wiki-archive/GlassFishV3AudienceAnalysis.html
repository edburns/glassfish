User Categories

The GlassFish Server product and its documentation are aimed at the following audience categories:

  • Developers using the Java[TM] language
  • Administrators
  • Deployers
  • Operators

Types of Corporations

Types of corporations that use GlassFish Server are as follows:

  • OEMs. Original equipment manufacturers with broad needs ranging from delivering core parts of GlassFish V3 in embedded devices such as network routers to bundling GlassFish V3 feature-rich capabilities as part of a larger enterprise solution.
  • ISVs. Independent software vendors that deploy to a platform the fits the requirements of the problem their product addresses.
  • ASPs. Hosting providers and enterprises that use virtualization technologies want to offer web container services without the overhead of unnecessary containers.
  • Enterprises. Businesses that aim to drive down costs and complexity for simple deployments but require feature-rich capabilities for mission critical deployments.
