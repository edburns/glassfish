GlassFish Administration Infrastructure Pages

The administration infrastructure area of GlassFish includes the following areas of the product:

  • CLI Infrastructure (command_line_interface JIRA component)
  • Configuration (configuration JIRA component)
  • Monitoring (monitoring JIRA component)
  • Clustering Management Infrastructure (distributed management JIRA component)
    • SSH/DCOM Node support
    • Command Replication 
    • Synchronization 
  • Domain Backup and Recovery 
  • Process Launcher
  • Admin Security  
  • OS Service Integration 
  • Miscellaneous Commands

All of the areas above that do not have a specific JIRA component are tracked with the admin JIRA component.
