Introduction

This is the overall master task list for all the tasks we would do for TP2. Refer to GUI V3 TP2 for what will be delivered. We will continue modify this list as we make progress and move on to the next milestone.

Besides the individual dependency listed for specific task, here is a general dependency list, some may apply for features after TP2 :

  • General Backend MBean Support
  • Logger/Log Level APIs are new, need support for this
  • API to Store arbitary data (Need MBean b/c of Remote MBean requirement)
  • Privileges (Grouped by Role)
  • Pluggability APIs
    • Support for finding plugins & their configuration
  • CRUD
    • Support for finding POJO / Meta-data from backend
    • Support from JSFTemplating
