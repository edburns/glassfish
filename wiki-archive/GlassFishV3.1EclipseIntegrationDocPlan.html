Audience

Developers using the Eclipse IDE.

Summary of the Feature From a User's Perspective

This feature integrates GlassFish 3.1 into the Eclipse IDE.

Availability

This feature is available in the GlassFish Tools Bundle for Eclipse download, the GlassFish Java EE Application Server Plugin for Eclipse download, and through the "Download additional server adapters" feature of Eclipse.
