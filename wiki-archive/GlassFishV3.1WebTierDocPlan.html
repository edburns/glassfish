Audience

Developers of Java EE application use the various APIs provided by the Web Tier services.

GlassFish administrators configure and manage the services provided by the Web Tier.

Summary of the Feature From a User's Perspective

The Web Tier provides the required Java EE 6 functionality for the web container.

Availability

The Web Tier is available in both the web and full GlassFish Server distributions.

Statement of Work

This section describes in detail for each document the work that is required by the introduction of the feature, such as changes, addition of new material, relocation of existing material, and removal of existing material.

Note
