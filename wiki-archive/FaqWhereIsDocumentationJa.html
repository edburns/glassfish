ドキュメントはどこにありますか?

企業の開発者、ユーザー、デプロイヤと管理者向けに、十分な高品質なドキュメントが、 GlassFish v2 と GlassFish v2 UR1 によって、自由に閲覧できます。

GlassFish v3 は、現在開発中ですが、 preliminary documentation については、マイルストーン最終版リリースの、 GlassFish v3 Technology Preview 2 (TP2) にて利用可能です。

The GlassFish v2 の文書は、サンの GlassFish 商用版である、 Sun Java System Application Server (SJSAS) として掲載されています。

GlassFish v2 は、 SJSAS 9.1 に対応し、そして、 GlassFish v2 UR1 は、 SJSAS 9.1 Update 1 に対応しています。

GlassFish v2 と SJSAS 9.1 を比較するには、 GlassFish v2 は、 Sun Java System Application Server 9.1 とは何が違うのか ? を見ると良いです。

GlassFish についての付加的な情報のドキュメンテーションと、貢献方法については、 ドキュメンテーションホームページ と コミュニティドキュメント のページを見てください。

その他のリソースはどこにありますか ?

スクリーンキャストインデックス 、 ティップスとブログのページ 、記事のページ, アクエリウム 、 ディスカッションフォーラム 、 メーリングリストアーカイブズ 、 そして チャットルーム から、付加情報を利用できます。

GlassFish ポッドキャスト の件数が増大している GlassFish コミュニティからのインタビューやニュース記事についても、利用することができます。

ドキュメントに関連して起きていることに、取り残されないためには、 Application Server Docs blog を訪れるべきでしょう。

ドキュメントに関連する議論に対して質問やコメントするためには、メーリングリスト [docs@glassfish.java.net] を利用する必要があります。

GlassFish メーリングリスト のページから、一覧へ投稿することができます。

どうやって GlassFish ドキュメンテーションへ貢献しますか ? を見てください。

ドキュメンテーションの wiki などはありますか ?

GlassFish トラブルシューティングガイド が編集可能な wiki フォーマットとして利用可能です。

また一方で、役立たせたり、最新版への改善の為に、彼らは、あなたの意見を必要としています !

あなたのコメント、提案、そしてソリューションで、貢献してください。
