OS/Platform Issues

OS/プラットフォームの制約

本セクションでは、下位の OS もしくは JDK に関する既知の制約事項やバグを載せています。

Java Development Kit (JDK) 1.5.x

異なるバージョンの JDK を使用する場合、互換性の問題がいくつかあります。

  • マイグレーションツールを動かすには JDK が必須です。JRE だけでは不十分です。
  • マイグレーションツールは JDK 1.5.x を必要とします。

Windows 2000 Service Pack 2

マイグレーションツールは Windows 200 の場合、Service Pack 2 もしくはそれ以降を必要とします。SP2 より以前のものが使用されると、入力パネルにファイルを表示する際、おかしな挙動をすることがあります。

ファイルチューザーの機能

出力ディレクトリを選択する際ポップアップするファイルチューザーダイアログは、JDK 内の機能の欠陥によりいくつかの機能を持ちません。「フォルダの新規作成」ボタンは新規フォルダを作成しますが、新しく作成されたフォルダをハイライトしません。このことで混乱するユーザがいるかも知れません。

フロッピードライブ (A を選択する際の問題

Windows にて入力もしくは出力ディレクトリを選択する際、フロッピードライブ (A:) をクリックすると、(ドライブが空の場合には) 「java.exe - No Disk」という名前で、Abort, Retry, Ignore の 3 つのボタンのあるダイアログがポップアップします。このポップアップは Ignore ボタンをクリックしても 2 度表示されます。

入力での JAR, WAR, RAR, EAR のファイル名の問題

マイグレーションツールは入力として、ファイル名にスペースを持つものを受け付けません。
