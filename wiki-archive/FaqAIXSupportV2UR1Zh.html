AIX 上是否支持 GlassFish v2 UR1？

支持。GlassFish v2 UR1（单击此处下载）是第一个在 AIX（AIX 5.3 64 位 OS 和 AIX 5.2 64 位 OS，在 32 位 JVM 上）上受支持的版本。仅支持开发者和群集配置文件。

发行说明中列出了 GlassFish v2 UR1 发行版（相当于 Sun 的商业产品 Sun Java System Application Server 9.1 Update 1）中支持的平台。

注意：有关 GlassFish 和 Sun 的商业 Application Server 的差别的详细描述，请参见 GlassFish V2 和 Sun Java System Application Server 9.1 的差别是什么？
