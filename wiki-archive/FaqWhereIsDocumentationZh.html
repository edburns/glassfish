文档位于何处？

GlassFish v2 和 GlassFish v2 UR1 的完整的企业级开发者、用户、部署者和管理员文档都是免费提供的。

GlassFish v3 目前正在开发中，不过可以提供最新里程碑发行版 GlassFish v3 技术预览 2 (Technology Preview 2, TP2) 的预备文档。

GlassFish v2 文档被列为 Sun Java System Application Server (SJSAS) ，它是 Sun GlassFish 的商业版本。GlassFish v2 对应于 SJSAS 9.1，GlassFish v2 UR1 对应于 SJSAS 9.1 Update 1。有关 GlassFish v2 和 SJSAS 9.1 之间的详细比较，请参见 GlassFish v2 与 Sun Java System Application Server 9.1 有何不同？。

有关 GlassFish 文档以及如何参与讨论的附加信息，请参见文档主页和社区文档页面。

还有哪些其他资源可供使用？

可以从截屏视频 (screencast) 索引、提示和博客页面、文章页面、The Aquarium、讨论论坛、邮件列表归档和聊天室获得其他信息。此外，GlassFish 播客也越来越多，提供了 GlassFish 社区的访谈和新闻。

要随时掌握与文档相关的事件，请访问应用服务器文档博客。要讨论与文档相关的任何内容以及您的任何问题或意见，请使用 [docs@glassfish.java.net] 邮件列表。您可以通过 GlassFish 邮件列表页面订阅列表。

如何参与讨论？

请参见如何为 GlassFish 文档作贡献？。

是否有文档 Wiki？

GlassFish 疑难解答指南是采用完全可编辑的 Wiki 格式提供的。但是，要想让它们成为有用的资源而且尽可能保持最新，需要您的输入！请提出您的意见、建议和解决方案。
