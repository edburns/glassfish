GlassFish V3 Downloads Page

This page contains content pertaining to downloads of the v3 GlassFish Application Server.

Current Release

What is it?

The current release is the Technology Preview (TP) 2 Release. This technology preview allows you to experiment with a very early version of the GlassFish V3 app server. Note: This software is feature-incomplete, has undergone no formal testing, and is not suitable for production deployments. Even so, you might find it interesting to experiment with this code to experience the highly modular and lightweight approach we are taking in V3.

Note: the majority of the GlassFish code is available under a Dual License consisting of the Common Development and Distribution License (CDDL) v1.0 and GNU General Public License (GPL) v2 The following page contains details about the components in GlassFish and the licenses under which they are covered.

How do I get it?

See the Technology Preview (TP) 2 page for the current download link to get the binary release for all platforms. Note, some browsers may require you to right-click/CTRL-click to save these files to your computer.

How do I install it?

  1. Download the binary release
  2. cd to the directory into which you want to install the release
  3. Run:
    _unzip filename.zip_
    where filename.zip is the name of the release zip file. This command installs this release into a new GlassFish directory under your current default directory.

How do I get started?

Go to the GlassFish v3 Users page for more information.
